# TalkingGameSite
This is the official website for my RPG, which I call the Talking Game. It is currently available only if downloaded and run through the designated site.

This is going to be the official, one of a kind Talking Game website. It is made for the display and organization of information relating to the Talking Game.

In this website I include several features:

- All data about the game, including characters, worlds, magic, etc.
- Character sheets the players have made, and how to update them.
- Images, such as maps, character art, etc.
- "Secret" information, that you can access with passwords found in-game.
